Greenerizer

Project ini merupakan project dalam grup. Saya bekerja sebagai Mobile Developer menggunakan bahasa pemograman Kotlin. Aplikasi ini berbasis Android, aplikasi ini berguna untuk menghubungkan pihak pengelola sampah dengan masyarakat yang ingin mendaur ulang sampah. Project ini saya kerjakan sebagai syarat kelulusan dalam Bangkit 2023 Batch 1.

Identifikasi Volume Sampah Dengan Metode Convolutional Neural Network

Project ini merupakan tugas akhir sebagai syarat mendapatkan gelar Sarjana Teknik. Peran saya dalam project ini berguna untuk mengumpulkan dataset untuk Machine Learning, mengintegrasikan Machine Learning dengan kamera. Saya mengagaskan topik ini dikarenakan saya melihat masalah lingkungan di Indonesia ini masih harus menjadi perhatian. Saya melihat bahwa masalah lingkungan terutama sampah di sungai dapat diselesaikan dengan bantuan teknologi. Apalagi saat ini, perkembangan teknologi terutama Machine Learning dapat membantu menyelesaikan masalah lingkungan ini.
